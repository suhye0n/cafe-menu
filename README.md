# 카페 메뉴관리

### 기술 스택
- 프론트엔드: Vue.js
- 백엔드: Spring, MySQL

### 기능
**배너 관련**
- 2초 슬라이딩
- <, > 슬라이딩
- 하단 버튼 슬라이딩

**상품 주문**
- 주문 가능한 상품 리스트
- 장바구니 추가
- 주문하기

**상품 주문 내역**
- 상품 주문 리스트
- 주문 처리/취소
